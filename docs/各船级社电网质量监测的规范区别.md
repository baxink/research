# 各船级社电网质量监测要求对比分析（修订版）

> **核心参考来源**：
> - CCS：《钢质海船入级规范》2024版 第4篇（[ccs.org.cn](https://www.ccs.org.cn/ccswz/specialDetail?id=202406120326276668)）
> - DNV：Rules for Classification, Pt.4 Ch.8 (July 2024 edition)（[rules.dnv.com](https://rules.dnv.com/servicedocuments/dnvpm/)）
> - LR：Classification Alert 34/2017 – Harmonic Distortion（[lr.org](https://www.lr.org/en/knowledge/class-news/34-17/)）
> - ABS：Guidance Notes on Control of Harmonics in Electrical Power Systems（[eagle.org](https://ww2.eagle.org/content/dam/eagle/rules-and-guides/current/conventional_ocean_service/150_ctrlofharmonicsinelecpowersystems/electrical-harmonics-gn.pdf)）；Requirements for Hybrid-Electric Power Systems 2024
> - BV：NR467 Rules for Classification, Pt.C Ch.2
> - IACS：Requirements concerning Electrical and Electronic Installations（[2023版](https://safety4sea.com/wp-content/uploads/2023/10/IACS-Requirements-concerning-electrical-and-electronic-installations-2023_09.pdf)）
> - 通用基准标准：**IEC 61000-2-4**（工业环境兼容等级）、**IEEE 519**

## 一、强制性 vs. 建议性要求对比

### 1.1 总览表

| 对比维度 | CCS | DNV | LR | ABS | BV |
|:---|:---|:---|:---|:---|:---|
| **基础规范** | 《钢质海船入级规范》2024 第4篇 第3章 | Pt.4 Ch.8 Sec.4 (2024) | Pt.6 Ch.2; Alert 34/2017 | Pt.4 Ch.8; GN on Harmonics; Hybrid-Electric Reqts 2024 | NR467 Pt.C Ch.2 |
| **谐波分析报告** | **强制**（非线性负载占比超限时） | **强制**（所有含VFD的船舶） | **强制**（Alert 34/2017明确） | **强制**（GN中明确要求） | **强制**（所有电力推进船） |
| **PQM在线监测系统** | 建议性（附加符号时强制） | 附加符号时**强制** | 建议性（附加符号时强制）| 建议性（IPS/Hybrid时强制）| 建议性（附加符号时强制）|
| **海试谐波实测** | 建议性 | EP/CLEAN符号**强制** | SEA(EP)符号**强制** | IPS/Hybrid系统**强制** | COMF符号**强制** |
| **滤波器设计审核** | 有条件强制（超限时） | **强制**（超限时需安装并审核）| **强制**（Alert 34/2017重申）| **强制**（GN中有明确指导）| 有条件强制 |

### 1.2 关键差异解读

#### ① IACS统一要求（2023版）——各社共同底线

根据 IACS E5/E24 相关要求：
- **所有IACS成员社**均要求船舶电力系统中的非线性负载不得导致供电质量严重劣化。
- 但IACS层面**未设定具体THD数值限值**，各社自行细化。
- 这是各社限值**高度趋同但不完全一致**的根源。

#### ② 各社差异化立场

```
强制程度排序（从严到松）：

  DNV ▓▓▓▓▓▓▓▓▓▓ 最严（基本入级即强制谐波分析+文件审批）
  ABS ▓▓▓▓▓▓▓▓▒▒  次严（专项GN文件体系完善，IPS/Hybrid要求高）
  LR  ▓▓▓▓▓▓▓▒▒▒  Alert 34/2017后显著加严
  BV  ▓▓▓▓▓▓▒▒▒▒  附加符号体系驱动
  CCS ▓▓▓▓▓▒▒▒▒▒  基本入级中灵活度较大，附加符号时收紧
```

> **LR特别说明**：LR于2017年发布 **Classification Alert 34/2017**（[参考链接](https://www.lr.org/en/knowledge/class-news/34-17/)），明确要求：
> - 含谐波滤波器的船舶，**须提交完整的谐波分析报告**作为图纸审批条件；
> - 谐波滤波器本身须进行**型式试验或产品认可**；
> - 滤波器安装后须验证其实际滤波性能与设计一致。
>
> 这是LR在谐波管控领域从"建议性"转向"强制性"的标志性文件。

## 二、附加符号对电网质量的特殊要求

### 2.1 DNV 附加符号体系（业界最完善）

| 附加符号 | 与电网质量的关联 | 特殊要求摘要 |
|:---|:---|:---|
| **EP**（Electrical Propulsion） | **直接关联** | ① 强制提交所有典型工况谐波分析；② 主汇流排THDu严格限制（通常≤5%）；③ 海试实测验证为强制项 |
| **RP**（Redundant Propulsion） | **直接关联** | 在EP基础上增加：**冗余降级模式**下的电网质量评估（如一台发电机故障后剩余系统THD是否仍达标） |
| **CLEAN DESIGN** | **间接关联** | ① 环保设备（如SCR、BWTS控制系统）不得因谐波污染误动作或性能下降；② 要求验证EMC兼容性；③ 推进VFD谐波控制纳入CLEAN要求体系 |
| **DYNPOS**（DP） | **直接关联** | ① DP控制系统传感器对电压畸变**极敏感**；② 要求谐波不引起DP参考系统（MRU/DGPS等）误差；③ DP-3等级可能强制PQM |
| **BATTERY / HYBRID** | **直接关联** | 电池BMS的DC/AC变换器、充电整流器引入的谐波需**专项评估**；与ABS Hybrid 2024要求趋同 |
| **SMARTSHIP** | **数据层关联** | 电网质量参数（V、I、P、Q、PF、THDu、THDi）须接入船舶数据中心，支持远程查看与趋势分析 |

### 2.2 ABS 特殊体系

根据ABS **Guidance Notes on Control of Harmonics in Electrical Power Systems**（[原文PDF](https://ww2.eagle.org/content/dam/eagle/rules-and-guides/current/conventional_ocean_service/150_ctrlofharmonicsinelecpowersystems/electrical-harmonics-gn.pdf)）：

> ABS是各大船级社中**唯一发布独立谐波控制指导文件**的机构。此GN文件包含：
> - 谐波产生机理与建模方法的系统指导
> - 滤波器选型与设计的工程指南
> - **同时引用 IEC 61000-2-4 和 IEEE 519**，对电压和电流谐波设双重限值
> - 非线性负载的典型谐波频谱特征数据库

此外，ABS **Requirements for Hybrid-Electric Power Systems 2024**（[原文PDF](https://ww2.eagle.org/content/dam/eagle/rules-and-guides/current/conventional_ocean_service/319-requirements-for-hybrid-electric-power-systems-for-marine-and-offshore-applications-2024/319-hybrid-electric-power-systems-reqts-Apr24.pdf)）新增：
- 混合动力系统中**多变流器并联运行**时的谐波叠加分析要求
- 电池系统充放电切换过程中的**暂态电能质量**评估

### 2.3 各社附加符号简要对比

| 船级社 | 核心附加符号 | 对电网质量的附加要求强度 |
|:---|:---|:---|
| **DNV** | EP / RP / CLEAN / DYNPOS / BATTERY | ★★★★★ 最全面，各符号均有细化条文 |
| **ABS** | IPS / Hybrid-Electric / ACCU | ★★★★☆ 独立GN最详尽，Hybrid 2024版新增内容多 |
| **LR** | SEA(EP) / ShipRight ECO / ICE | ★★★☆☆ Alert 34/2017后显著强化 |
| **BV** | COMF-V / COMF-C / CLEANSHIP | ★★★☆☆ 以乘客舒适度为切入点间接管控 |
| **CCS** | i-Ship(E) / Green Ship / Smart Ship | ★★☆☆☆ 基本入级灵活，附加符号时才收紧 |

## 三、谐波限值差异化规定

### 3.1 电压总谐波畸变率（THDu）

| 场景 | CCS | DNV | LR | ABS | BV |
|:---|:---|:---|:---|:---|:---|
| **基本入级（一般船舶）** | ≤ 8% | ≤ 8% | ≤ 8% | ≤ 8% | ≤ 8% |
| **电力推进船（附加符号）** | ≤ 5% | ≤ 5% | ≤ 5% | ≤ 5% | ≤ 5% |
| **敏感设备端子处** | — | ≤ 5%（EP） | ≤ 5% | ≤ 5% | ≤ 5%（COMF） |
| **DP系统关联配电板** | — | ≤ 5%（DYNPOS） | 参照DP规范 | — | — |
| **依据标准** | IEC 61000-2-4 | IEC 61000-2-4 | IEC 61000-2-4 / IEEE 519 | **IEEE 519 + IEC 61000-2-4**（双轨） | IEC 61000-2-4 |

> **关键发现**：THDu数值限值各社**完全一致**（8%/5%两档），因均源自IEC 61000-2-4 Class 2/Class 3边界值。

### 3.2 真正差异所在——六大维度

各社谐波限值的**核心差异不在数值，而在适用条件和执行深度**：

| 差异维度 | 最严者 | 具体差异 |
|:---|:---|:---|
| **① 考核工况数量** | **DNV** | DNV EP：须覆盖**所有典型运行工况**（港口/航行/DP/应急等）。CCS：重点关注"最恶劣谐波工况"即可 |
| **② 考核点位置** | **DNV / BV** | DNV：主汇流排 **+** 敏感设备端子。BV(COMF)：主汇流排 **+** 旅客区域配电板。CCS/LR：以主汇流排为主 |
| **③ 电流谐波THDi** | **ABS** | ABS GN引用IEEE 519，**同时限制电压THD和电流THD**（按ISC/IL比值分档）。其余四社以电压THD为主要管控指标 |
| **④ 谐波频率覆盖** | **DNV** | DNV：要求评估至**50次谐波**。CCS/LR/BV：一般要求至**40次**。ABS(GN)：建议至50次 |
| **⑤ 间谐波评估** | **DNV** | DNV EP/BATTERY：要求对变流器产生的**间谐波**（interharmonics）进行单独评估。其余社一般不做单独强制要求 |
| **⑥ 暂态电能质量** | **DNV / ABS** | DNV DYNPOS：暂态电压偏差≤±20%，恢复时间≤1.5s。ABS Hybrid 2024：电池切换暂态的电压/频率偏差有明确限值 |

### 3.3 ABS 电流谐波限值（独特之处）

ABS GN引用 **IEEE 519-2014**，在电压THD之外增加了**电流THD分档限值**：

| ISC/IL比值 | 奇次谐波电流限值（占基波%） | THDi |
|:---|:---|:---|
| < 20 | 3次~11次: 4.0% / 11次~17次: 2.0% | 5.0% |
| 20~50 | 3次~11次: 7.0% / 11次~17次: 3.5% | 8.0% |
| 50~100 | 3次~11次: 10.0% / 11次~17次: 4.5% | 12.0% |
| 100~1000 | 3次~11次: 12.0% / 11次~17次: 5.5% | 15.0% |
| > 1000 | 3次~11次: 15.0% / 11次~17次: 7.0% | 20.0% |

> **ISC** = 短路电流，**IL** = 最大需求负载电流。此为ABS区别于其他四社的**最显著特征**。
>
> ⚠ **验船师注意**：当船舶同时持有ABS和其他船级社入级时，需额外进行电流谐波评估，设计方可能遗漏此项。

## 四、实务检验建议

### 4.1 图纸审查核查清单

```
□ 是否提交谐波分析报告？
    → 检查适用条件（VFD容量占比是否触发强制要求）
□ 谐波分析工况是否覆盖完整？
    → DNV EP：所有典型工况  |  CCS：至少含最恶劣工况
□ 计算模型参数是否合理？
    → VFD脉冲数（6/12/18/24脉冲）与实际选型一致？
    → 电缆阻抗是否按实际路径长度建模？
    → 发电机次暂态电抗参数来源是否可靠？
□ 限值选用是否正确？
    → 基本入级8% or 附加符号5%？
    → ABS船：是否同时校核电流THD？
□ 滤波器方案：
    → 有源 or 无源？调谐频率是否覆盖主要谐波次数？
    → 多台滤波器并联时是否考虑谐振风险？
```

### 4.2 海试/在船检验要点

| 项目 | 要求 | 现场注意事项 |
|:---|:---|:---|
| **仪器精度** | IEC 61000-4-30 **Class A** | ⚠ 部分船厂或设备商自带仪器可能仅为Class S，不满足要求 |
| **测试工况** | 须在实际运行负载下 | ⚠ 码头系泊试验中推进器无法满功率运行，部分工况可能需在海试中补测 |
| **测点选择** | 主汇流排PT二次侧 | ⚠ 注意PT精度等级及变比误差对测量结果的影响 |
| **采样时间** | ≥ 10分钟稳态 | ⚠ 短时测量可能遗漏间歇性谐波源（如货泵启停） |
| **环境因素** | — | ⚠ 机舱温度高、噪音大、照明不足、接线空间狭小——建议提前与船厂确认测试接入点的可达性 |
| **结果判定** | 实测THDu ≤ 限值 | 若实测值超过计算值20%以上，需重新审视计算模型 |

### 4.3 常见缺陷与处理

| 常见问题 | 原因分析 | 处理建议 |
|:---|:---|:---|
| 实测THDu超过8% | VFD脉冲数不足（6脉冲多发）、滤波器未投入 | 要求安装/调整有源滤波器；或升级VFD为多脉冲/多电平 |
| 计算报告中工况不完整 | 设计方仅计算"正常航行"工况 | 退审，要求补充最小发电+最大非线性负载工况 |
| 滤波器型式试验证书缺失 | LR Alert 34/2017后常见 | 要求提交滤波器产品认证或型式试验报告 |
| 不同变频器间谐波谐振 | 多台VFD并联且调谐频率接近 | 要求提交多VFD并联谐振分析 |

## 五、总结矩阵

| 维度 | CCS | DNV | LR | ABS | BV |
|:---|:---|:---|:---|:---|:---|
| **THDu限值** | 8%/5% | 8%/5% | 8%/5% | 8%/5% | 8%/5% |
| **THDi限值** | 未单独规定 | 未单独规定 | 未单独规定 | **有（IEEE 519）** | 未单独规定 |
| **强制谐波分析** | 有条件 | **全面强制** | 强制（Alert后） | 强制（GN） | 强制（电推船） |
| **附加符号完善度** | ★★ | ★★★★★ | ★★★ | ★★★★ | ★★★ |
| **独立谐波指导文件** | 无 | 无（但Sec.4详尽） | Alert 34/2017 | **有（独立GN）** | 无 |
| **间谐波要求** | 无 | **有（EP/BATTERY）** | 无 | 建议性 | 无 |
| **海试实测要求** | 建议性 | 附加符号**强制** | 附加符号强制 | IPS/Hybrid**强制** | 附加符号强制 |

### 一句话总结

> **数值上各社一致（均基于IEC 61000-2-4），差异在执行深度（DNV最全面）和电流谐波管控（ABS独有IEEE 519双轨制）。** 多船级社并行项目应取各项最严值为设计目标。
